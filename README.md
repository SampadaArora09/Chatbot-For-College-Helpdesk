Chatbot-For-College-Helpdesk
A simple chatbot built using Python, NLTK, and Streamlit that answers basic college-related queries.

Live App
 [Click here to try the chatbot] (https://chatbot-for-college-appdesk-mjh6ysrttgxxuj7hnomydb.streamlit.app/)

 Tech Used
- Python  
- NLTK  
- Streamlit  

 Files
- `college_chatbot_streamlit.py` — Main Streamlit app  
- `requirements.txt` — Dependencies  

How to Run Locally
pip install -r requirements.txt
streamlit run college_chatbot_streamlit.py
